INSTALLATION
  1. Upload all files in this package while maintaining the directory structure (rename YOUR_ADMIN to your custom admin folder name)
  2. Go to ADMIN->TOOLS->IMPORT COUPONS to use the tool.
TIPS
  • Always backup your database (specifically the tables related to coupons) before importing a list. You may want to revert your database from the backup and make adjustments before importing again.
  • The generic CSV format uses one field per coupon (i.e. 12345,67890,01234)
  • The Groupon CSV format is provided by Groupon after the sale has ended, or as sales are made. Check with your Groupon representative for clarification.
UNINSTALL
  1. Delete all uploaded files from your store.
  2. If using Zen Cart v1.5.0 or later, patch your database by copying and pasting the contents of uninstall.sql into ADMIN->TOOLS->INSTALL SQL PATCHES
